สรุปแล้วทั้ง Git Kraken แล้ว SourceTree นั่นเป็น VC (Version Control) เหมือนกันทั้ง 2 Software นี้ ซึ่งได้ออกแบบหน้า GUI เพื่อให้ USER ใช้งานได้ง่ายและสะดวกมากยิ่งขึ้น และก็อย่างที่ทราบคือ ประโยชน์ของมันนั้นสามารถเก็บไฟล์ Back Up ขึ้น GitHub ได้ และเป็นเป็น Version ไปนั่นเองซึ่งมันทำให้เกิดผลดีต่อ Programmer มากเพราะไม่ต้องมานั่งจำว่าเห้ยเราทำผิดตรงไหนน เฮ้ยเราลืมเซฟไฟล์ไว้ จะแก้กลับไปยังไง และสุดท้ายนี้หากมีข้อผิดพลาดประการใดก็ขออภัยมา ณ ที่นี้ด้วยนะครับ
